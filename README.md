# # 四、**快速开始（安装 & 使用**)

## 环境要求

- Node.js ≥ ^20.19.0 || >=22.12.0（推荐 22.18.0 版本）
- yarn ≥ 1.22.22
- Cesium ≥ 1.108.0（项目内置版本为 1.110.0，兼容 1.108+）
- 浏览器：Chrome ≥ 90 / Firefox ≥ 88（需支持 WebGL 2.0）



## 1、初始化项目
```cmd
npm init vite@latest vue-app-template -- --template vue
```
